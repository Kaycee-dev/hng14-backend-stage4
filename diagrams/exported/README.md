# Exported Diagrams

Embeddable diagram artifacts for Google Docs live here.

## Active Artifacts

- `architecture.md` - export-ready reference for the architecture image: source path, render command, embed checklist, and the component list the rendered image must contain.
- `architecture.png` - rendered image (created locally via the export workflow described in `architecture.md`; placed here before the Google Docs embed step).

## Rules

- Exports must match `diagrams/source/architecture.mmd`. If the source changes, re-export.
- The embedded image in the Google Docs submission must show the simplified core path and a clearly separated conditional scale-levers box. Anything else means the diagram has drifted from the final submission design.
